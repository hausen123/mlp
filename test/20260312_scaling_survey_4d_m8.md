# スケーリングサーベイ（4d FFN, M=8） — 独立性クエリ品質 vs データ量

## 目的
FFN中間層4d・M=8のMLPMemoryで、M=22と同様のスケーリング実験を行い
品質劣化閾値を比較する。

## 変更点
- FFN: Linear(d, 4d)（mlp.py MLPBlock）
- M=8（num_layers=8）

## データ準備
- 元JSONL: `data/202510301810_250304_kangaekata_rag_workflow.jsonl`（273,516件）
- `data/wo_independence.jsonl`: 元JSOLから「原子力規制委員会」AND「独立性」を含む行を除外（272,139件）
- `data/independence.jsonl`: 1,377件（固定）

## 学習コマンド
```
pipenv run python mlp.py \
  --mode qa-train \
  --qa_path data/mixed_stepN.jsonl \
  --epochs 10 \
  --num_layers 8 \
  --use_final_layer \
  -m "4d M=8 scaling step N: independence(1377) + wo(XXXX)"
```

## 推論コマンド（λ=0.8, temperature=0.2）
```
pipenv run python mlp.py \
  --mode infer \
  --model_dir model/YYYYMMDDHHMM_qwen25-05b-instruct-qa \
  --lambda_interp 0.8 \
  --temperature 0.2 \
  --skip_base_lm \
  --prompt "原子力規制委員会の独立性について説明してください。"
```
